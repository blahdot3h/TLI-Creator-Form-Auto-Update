# TLI-Creator-Form-Auto-Update
Auto Update Scripts for TLI Creator Form

Open your copy of the template form on **Google Sheets** and select Extensions -> Apps Scripts

<img width="288" height="154" alt="image" src="https://github.com/user-attachments/assets/a4f37c75-a1db-40fc-9e62-114b5b7479e4" />

Once inside of the Apps Scripts Add the services for

**YouTube**

**YouTube Analytics**

<img width="242" height="160" alt="image" src="https://github.com/user-attachments/assets/830f5fe2-3a87-46ee-a944-b4b958ee4a81" />

Add Files and paste the Contents of YouTubeParse into a new script named YoutubeParse.gs into your Files

Click on the Settings Icon and Add These Script Properties

Login to https://dev.twitch.tv/ to get your CLIENT_ID and CLIENT_SECRET

From within your Console click Register Your Application

<img width="819" height="86" alt="image" src="https://github.com/user-attachments/assets/a0b76f1b-12f2-42d9-aa98-da00966eab0b" />

Create your Application with the redirect URL just set to localhost and save your CLIENT_ID and CLIENT_SECRET
<img width="1051" height="759" alt="image" src="https://github.com/user-attachments/assets/d99d37b0-d623-45e1-aafe-fff09403acec" />


<img width="255" height="305" alt="image" src="https://github.com/user-attachments/assets/e25bf031-7f77-450d-b2ca-08137a8f8aa2" />


<img width="884" height="635" alt="image" src="https://github.com/user-attachments/assets/8838cb96-0073-465b-b024-d7cb0356efcf" />

CLIENT_ID is pulled from [Twitch Dev](https://dev.twitch.tv/)
CLIENT_SECRET is pulled from [Twitch Dev](https://dev.twitch.tv/)
CLIENT_USERNAME is your Twitch name

Under the Triggers Tab select Add Trigger
Set it to Trigger On Open and set this for each Script

<img width="769" height="671" alt="image" src="https://github.com/user-attachments/assets/fa40e052-53c4-451d-92ef-13a07f8024a2" />

You should see both triggers under here now
<img width="817" height="173" alt="image" src="https://github.com/user-attachments/assets/858b7382-9ff7-4fb8-ad8f-6aa304c6d981" />

Reload your sheet and you should see it automatically update and pull in your information
